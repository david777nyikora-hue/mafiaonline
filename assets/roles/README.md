# Imagini pentru Roluri

Acest folder poate conține imagini pentru fiecare rol în joc.

## Imagini Recomandate

Poți adăuga următoarele imagini (dimensiune recomandată: 256x256px, format PNG):

- **mafia.png** - Icon pentru Mafia (ex: armă, mască, gangster)
- **doctor.png** - Icon pentru Doctor (ex: stetoscop, cruce medicală)
- **detective.png** - Icon pentru Detectiv (ex: lupă, pălărie detectiv)
- **citizen.png** - Icon pentru Cetățean (ex: persoană simplă, om de rând)

## Surse pentru Imagini Gratuite

- [Flaticon](https://www.flaticon.com/) - Icoane gratuite
- [Freepik](https://www.freepik.com/) - Grafică gratuită
- [Icons8](https://icons8.com/) - Icoane și ilustrații
- [Unsplash](https://unsplash.com/) - Fotografii gratuite

## Notă

Jocul funcționează perfect și fără imagini, folosind emoji-uri în schimb (🔫 💊 🔍 👤).
Imaginile sunt complet opționale și pot fi adăugate pentru o experiență vizuală îmbunătățită.
